# 早盘要闻

开盘前（8:30）生成精简早报，回答一个问题：**今天开盘会怎么走？**

**核心原则：脚本拿数据（零幻觉），搜索拿新闻。Token 优先。只采集，不分析。**

## 数据采集架构

```
脚本（纯 HTTP，零浏览器）:
  scrape_pre_market.py ──→ 美股指数 / A50期货 / 黄金 / 原油 / 外汇 / 中概股
    ├── 新浪 hq.sinajs.cn ──── 黄金/原油/外汇/中概股（稳定，不限流）
    ├── 东财 push2.eastmoney ── 美股指数（实际值）/ A50期货（SGX）
    └── 新浪 ETF 代理 ──────── 东财不可用时：SPY/QQQ/DIA（涨跌幅=指数涨跌幅）

WebSearch（仅新闻，site: 限定）:
  ① CNBC 隔夜美股          site:cnbc.com
  ② 国际要闻                全网搜索
  ③ 财联社盘前              site:cls.cn
```

> 国内政策（国务院/央行等）不单独搜索——财联社盘前汇总已覆盖，且收盘日报的国内政策源已经是全网最全。早报只关注增量：隔夜美股、国际事件、盘前突发。

## 备用机制（Fallback）

### 脚本 fallback（scrape_pre_market.py 内置）

| 场景 | source 值 | 处理 |
|------|-----------|------|
| 全部东财+新浪正常 | `direct_api` | 正常使用 |
| 东财不可用，新浪 ETF 代理美股 | `partial_fallback` | 正常使用（ETF 涨跌幅=指数涨跌幅，**无需 ⚠️**） |
| A50 不可用 | `fallback_items: ["A50期货"]` | WebSearch 兜底，**标记 ⚠️** |
| 大部分数据缺失 | `fallback_needed` | 全部 WebSearch，**标记 ⚠️** |

### A50 期货兜底搜索

如果脚本输出中 `a50_futures` 为 `null`：

```
搜索词：富时中国A50指数期货 最新
```

> A50 是最重要的开盘方向指标。东财单次调用不易触发限流（我们之前的失败是因为密集测试），生产环境大概率直接成功。

### 新闻搜索 fallback

与 信息采集 相同规则：
- `site:` 返回 0 条 → `site:news.cn`（新华网）重试，**无需 ⚠️**
- `site:news.cn` 也返回 0 → 去 `site:` 自由搜索，**标记 ⚠️**

> 实际触发概率很低——CNBC 和财联社索引稳定。国际要闻全网搜索不需要 site:。

## 执行流程

### 第〇步：交易日判断

早盘要闻**只在 A 股交易日运行**（周一至周五，排除中国法定节假日）。

- 周末/节假日 → 直接告知用户"今日休市，无需早盘要闻"

### 第一步：运行脚本

```bash
python scripts/scrape_pre_market.py
```

1 个脚本，输出包含所有市场数据。

### 第二步：并行新闻搜索

| 来源 | 搜索词 | 条数 |
|------|--------|------|
| 隔夜美股 | `site:cnbc.com US stock market close YYYY年MM月DD日` | 3-5 |
| 国际要闻 | `YYYY年MM月DD日 国际要闻 地缘政治 全球市场 大宗商品` | 3-5 |
| 财联社盘前 | `site:cls.cn 盘前 YYYY年MM月DD日` | 3-5 |

### 第三步：检查脚本输出

1. 如果 `source == "partial_fallback"`：美股用 ETF 代理，正常使用（不需 ⚠️）
2. 如果 `a50_futures == null`：WebSearch 兜底搜 A50，标记 `⚠️ 备用数据源`
3. 如果 `source == "fallback_needed"`：全部 WebSearch 兜底

### 第四步：处理周一/节后场景

判断今天距上一交易日多少天：

```
用 Glob 扫 Obsidian 目录，取最新一份 A股市场日报_*.md 的日期，计算 gap
```

| gap | 场景 | 调整 |
|-----|------|------|
| 1 天 | 正常（周二-周五） | 无调整 |
| 2-3 天 | 周一 | 搜索词覆盖整个周末（`YYYY年MM月DD日` 替换为 `YYYY年MM月`），财联社多取几条 |
| > 3 天 | 节后 | 同上 + 搜索"假期重大事件" |

### 第五步：生成早报

按模板生成，写入 Obsidian。
