## 核心理念（不变）

```
                你的大脑                    ↑            AI负责辅助思考                    ↑      ┌──────────┼──────────┐      │          │          │   最新信息     深度研究      知识管理      │          │          │      └──────────┼──────────┘                 │             Obsidian

---

## 三层架构（不变）

### 第一层：信息（Input）

> 今天世界发生了什么？

### 第二层：研究（Research）⭐ 核心

> 为什么？为什么会涨？为什么政策转向？历史有没有类似？

**AI 最有价值的地方。**

### 第三层：知识库（Obsidian）

> 自己的理解。不是新闻，不是资讯。

## 早盘要闻（morning-brief）设计

### 时间线

```
T-1 15:00  A股收盘
    ↓
   夜间    美股交易（北京时间 21:30 → 次日 04:00）
   夜间    商品期货交易
   夜间    公司公告陆续发布
    ↓
T 08:30   运行 morning-brief → 生成早报
T 09:00   集合竞价开始
T 09:30   A股开盘
    ↓
T 15:00   A股收盘
T 15:00+  运行 daily-market-news → 生成日报
```

### 数据采集架构

```
脚本（零幻觉，结构化数据）:
  scrape_pre_market.py  →  美股指数 + A50期货 + 商品 + 外汇 + 中概股
  scrape_announcements.py →  关注个股公告（巨潮资讯网 cninfo）

WebSearch（新闻，site: 限定）:
  ① site:cnbc.com 隔夜美股收盘
  ② 全网搜索 国际要闻
  ③ site:gov.cn / site:news.cn 收盘后新出的国内政策
  ④ site:cls.cn 盘前要闻
```

### 数据源详解

**脚本层：**

| 数据 | 来源 | 备注 |
|------|------|------|
| 标普500/纳斯达克/道琼斯 | 东财全球指数 API | 收盘点位+涨跌幅 |
| A50 期货（SGX） | 新浪期货 `hf_XINA50` | A股开盘方向最重要指标 |
| COMEX 黄金 | 新浪/东财商品 | 避险情绪代理变量 |
| WTI 原油 | 新浪/东财商品 | 通胀/地缘代理变量 |
| 美元指数 (DXY) | 新浪外汇 | 全球资金流向 |
| 离岸人民币 (USD/CNH) | 新浪外汇 | 外资进出 A 股意愿 |
| 中概股（KWEB/BABA/JD/PDD） | 新浪美股 | 中概股表现 → A股情绪传导 |
| 关注个股公告 | 巨潮资讯网 cninfo.com.cn | A 股法定信息披露平台，JSON API |

**搜索层：**

| 来源 | 搜索词 | 条数 |
|------|--------|------|
| 隔夜美股 | `site:cnbc.com US stock market close YYYY年MM月DD日` | 3-5 |
| 国际要闻 | `YYYY年MM月DD日 国际要闻 地缘政治 央行 大宗商品` | 3-5 |
| 国内政策增量 | `site:gov.cn 要闻 YYYY年MM月DD日` | 2-3 |
| 财联社早报 | `site:cls.cn 盘前 要闻 YYYY年MM月DD日` | 3-5 |

> 国内政策只搜当天（overnight 新出的），不重复搜白天已覆盖的。人数不够时降级到 `site:news.cn`。

Skill 4 📋 knowledge             知识整理
Skill 5 📊 review                周报/月报
```

### 三个 Skill 的协作关系

```
8:30  morning-brief    →  早盘要闻（数据）
8:45  market-research  →  全日预测（预判）
         ↓
9:30-15:00  盘中
         ↓
15:00+ daily-market-news → 收盘日报（验证）
         ↓
       market-research  → 预测回顾 + 认知修正
         ↓
       market-research  → 深度研究（日报推荐的问题）
```

### Skill 3：market-research 双场景

**场景 1 — 全日预测**：早盘数据 + 补充搜索 → 7 模块预测（开盘方向/风格/板块强弱/主力资金/情景推演/关键变量/验证节点），收盘后逐项对照修正认知。

**场景 2 — 深度研究**：一个问题向下挖 5 层（行业→国际→中国→历史→人性），建立逻辑树而非碎片信息。每天一个问题即可。
